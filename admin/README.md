# Word Teacher Admin 教师管理后台

教师/管理员专用的后台管理系统，用于管理班级、学生、场景和查看学习进度。

## 📸 功能预览

### 仪表盘
- 数据概览卡片（学生数、教师数、班级数、今日练习）
- 跟读完成率和平均分统计
- 快速操作入口

### 班级 & 学生管理
- 班级列表、创建、编辑、删除
- 学生批量导入（Excel）
- 学生信息编辑（座位号、姓名、密码）
- 学生详情及练习记录查看

### 场景管理
- 对话场景和跟读场景管理
- 场景封面图上传/AI生成
- 跟读句子管理（支持AI自动翻译）

### 进步追踪
- 班级学习趋势图表
- 学生排行榜
- AI 学习报告生成

## 🔐 默认账号

| 角色 | 账号 | 密码 | 权限说明 |
|-----|------|------|---------|
| 管理员 | `admin` | `admin123` | 完全访问所有功能 |
| 教师示例 | `teacher001` | `123456` | 仅访问分配的班级 |

## ✨ 功能模块

### 📊 仪表盘 Dashboard
| 功能 | 说明 |
|-----|------|
| 数据卡片 | 学生总数、教师总数、班级总数、今日练习数 |
| 跟读统计 | 今日跟读完成数、完成率、平均分 |
| 快速入口 | 新建跟读场景、新建对话场景、管理班级 |

### 👥 教师管理 (仅管理员可见)
| 功能 | 说明 |
|-----|------|
| 教师列表 | 查看所有教师账号 |
| 新增教师 | 创建新教师账号 |
| 编辑教师 | 修改姓名、密码、管理员权限 |
| 删除教师 | 删除教师账号 |
| 分配班级 | 设置教师负责的班级 |

### 🏫 班级管理
| 功能 | 说明 |
|-----|------|
| 班级列表 | 显示班级名称、年级、学生数、负责教师 |
| 新建班级 | 创建新班级，设置名称和年级 |
| 编辑班级 | 修改班级信息，分配负责教师 |
| 删除班级 | 删除班级（需先移除学生） |
| 学生管理 | 进入班级学生列表 |

### 👨‍🎓 学生管理
| 功能 | 说明 |
|-----|------|
| 学生列表 | 按座位号排序，显示学号、姓名、练习次数 |
| 批量导入 | Excel 导入学生（学号、姓名、密码、座位号） |
| 下载模板 | 下载 Excel 导入模板 |
| 编辑学生 | 修改座位号、姓名 |
| 修改密码 | 重置学生密码 |
| 查看详情 | 查看学生练习记录和成绩 |
| 删除学生 | 删除学生及其所有练习记录 |

### 🎙️ 跟读记录
| 功能 | 说明 |
|-----|------|
| 记录列表 | 查看所有跟读练习记录 |
| 状态筛选 | 按已完成/未完成筛选 |
| 评分详情 | 发音分、流利度、完成度、努力分 |
| 学生信息 | 显示学生姓名和班级 |

### 📚 场景管理
| 功能 | 说明 |
|-----|------|
| 跟读场景 | 创建/编辑跟读练习（句子列表） |
| 对话场景 | 创建/编辑 AI 对话场景 |
| AI 翻译 | 自动翻译英文句子为中文 |
| AI 封面 | 自动生成场景封面图 |
| 可见性 | 设置场景对学生是否可见 |
| 难度等级 | 设置场景难度（基础/进阶/挑战） |
| 关键词 | 管理场景关键词(vocabulary) |
| AI 提示词 | 自定义对话场景的 AI 行为 |

### 📈 进步追踪
| 功能 | 说明 |
|-----|------|
| 趋势图表 | 班级整体学习趋势可视化 |
| 学生排行 | 优秀学员排行榜 |
| AI 总结 | 智能生成学习报告 |
| 导出报告 | 导出 Excel/PDF 格式报告 |

## 🔑 权限系统

### 管理员 (isAdmin = true)
| 模块 | 权限 |
|-----|------|
| 教师管理 | ✅ 完全访问 |
| 班级管理 | ✅ 所有班级 |
| 学生管理 | ✅ 所有班级的学生 |
| 跟读记录 | ✅ 所有学生的记录 |
| 场景管理 | ✅ 编辑/删除所有场景 |
| 进步追踪 | ✅ 所有班级 |

### 普通教师 (isAdmin = false)
| 模块 | 权限 |
|-----|------|
| 教师管理 | ❌ 不可见 |
| 班级管理 | ✅ 仅负责的班级 |
| 学生管理 | ✅ 仅负责班级的学生 |
| 跟读记录 | ✅ 仅负责班级学生的记录 |
| 场景管理 | ✅ 查看全部，编辑仅自己创建的 |
| 进步追踪 | ✅ 仅负责的班级 |

## 🚀 启动方式

```bash
# 安装依赖
pnpm install

# 开发模式
pnpm dev

# 构建生产版本
pnpm build
```

## 🛠️ 技术栈

- **框架**: React 18 + TypeScript
- **构建工具**: Vite
- **UI 组件库**: Ant Design 5
- **图标**: Lucide React
- **图表**: ECharts
- **请求库**: Axios
- **样式**: SCSS
